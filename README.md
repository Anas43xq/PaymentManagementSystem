# PaymentManagementSystem

**PaymentManagementSystem** is a Windows Forms application built with **.NET Framework 4.7.2** and **Visual Studio 2022**. It allows users to manage and track various payments (university and miscellaneous) and monitor course credit progress. The system features a layered architecture with data validation, reporting, and a user-friendly interface.

## Status

- **Payments**: Fully functional
- **Courses**: Under development

## Features

- Add, edit, delete, and retrieve payments
- Manage categories and currencies
- Generate reports and filter transactions
- Track course credit progress (partial)

## Requirements

- Visual Studio 2022
- .NET Framework 4.7.2
- SQL Server (local or remote)
